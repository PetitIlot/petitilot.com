# Images Petit Îlot

## Format des images

### Hero (home)
- **Nom** : `hero-home.jpg`
- **Format** : JPG ou WebP
- **Résolution** : 1920x1080px minimum
- **Ratio** : 16:9 ou plus large
- **Poids** : < 500KB

### Activités / Ressources
- **Format** : JPG ou WebP
- **Résolution** : 800x600px (ratio 4:3)
- **Poids** : < 200KB

### Livres
- **Format** : JPG ou WebP
- **Résolution** : 400x600px (ratio 2:3)
- **Poids** : < 150KB

## Optimisation

Utiliser un outil comme :
- [TinyPNG](https://tinypng.com/) pour compresser les JPG/PNG
- [Squoosh](https://squoosh.app/) pour convertir en WebP
- [ImageOptim](https://imageoptim.com/) (Mac) pour optimiser en batch

## Organisation

```
public/images/
├── hero-home.jpg          # Image hero de la page d'accueil
├── activities/            # Images des activités
├── books/                 # Couvertures de livres
└── games/                 # Images des jeux
```
